# libcamera_mon

libcamera-vid monitor tool, which parses log message to get FPS/bitrate etc on raspberry pi system.

# data flow

`libcamera-vid`  -- pipe --> `libcamera_mon`  -- proc --> `/proc/libcamera_proc`
